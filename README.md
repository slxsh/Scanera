## Installation 
* Clone the repo.
* Download OpenCV 4.1.0 Android SDK
* Copy contents of `OpenCV-android-sdk/sdk/native/libs` to `android/app/src/main/jniLibs`
* `npm i`
* `npx react-native run-android`
